## 新增命令

### 实用功能

| 命令        | 缩写   | 权限     | 说明                    |
| ----------- | ------ | -------- | ----------------------- |
| `SASDUMPER` | `SASD` | `Master` | 查看插件版本            |
| `SASTEST`   | -      | `Master` | 测试与 SAS 后台能否连接 |
| `SASON`     | -      | `Master` | 启用自动汇报            |
| `SASTOFF`   | -      | `Master` | 禁用自动汇报            |
